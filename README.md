# SmartLivingGuide.com# Build a Profitable, AdSense‑Safe Website on GitHub — Start to Finish

This guide walks you from **zero → live site → AdSense approval → monetization**, using **GitHub Pages**. It includes a **repository name**, **required pages**, **content plan**, and **inspection/approval checklist**.

---

## 1) Choose a Profitable, AdSense‑Safe Niche

**Good niches (high approval + steady traffic):**

* Smart living & home tips
* Personal finance basics (budgeting, saving, credit)
* RV & travel guides
* Tech how‑to & troubleshooting
* Health & wellness (non‑medical advice)

**We’ll use this niche example:**

> **Smart Living / Practical Guides**

## 2) Repository & Site Name (IMPORTANT)

**Repository name (recommended):**

```
smartlivingguide
```

**GitHub Pages URL:**

```
https://yourusername.github.io/smartlivingguide/
```

**Why this works:**

* Clean
* Brandable
* Not spammy
* AdSense‑friendly

---

## 3) Create the GitHub Repository

1. Go to **github.com** → Sign in
2. Click **New Repository**
3. Repository name: `smartlivingguide`
4. Public ✔
5. Check **Add README.md** ✔
6. Create repository

---

## 4) Enable GitHub Pages (FREE Hosting)

1. Repo → **Settings**
2. Click **Pages**
3. Source:

   * Branch: `main`
   * Folder: `/root`
4. Save

⏱ In 1–3 minutes your site is live.

---

## 7) Homepage Content (index.html)

Your homepage must:

* Clearly explain **what the site does**
* Link to articles
* Look legitimate

**Homepage Sections:**

* Hero headline (who it’s for)
* Short mission statement
* Featured articles (links)
* Footer with Privacy & Terms

---

## 8) About Page (Very Important)

AdSense wants to know **who you are**.

Include:

* Purpose of the site
* Who it helps
* How content is created
* Transparency

Example:

> SmartLivingGuide provides practical, research‑based guides to help people save money, improve their homes, and make smarter daily decisions.

---

## 9) Contact Page

Required.

Include:

* Contact email (Gmail is fine)
* Optional form

Example:

```
Email: smartlivingguide@gmail.com
```

---

## 10) Privacy Policy (NON‑NEGOTIABLE)

Must include:

* Cookies
* Google AdSense
* Third‑party ads

Must explicitly mention **Google advertising cookies**.

---

## 11) Terms & Conditions

Include:

* No liability
* Informational purposes only
* No guarantees

---

## 12) AdSense‑Safe Article Requirements

Each article must:

* 800–1,500 words
* Original (NOT copied)
* Helpful, not salesy
* No illegal, adult, or misleading content

## 15) Design Rules (Keep It Simple)

AdSense prefers:

* White background
* Easy‑to‑read fonts
* Mobile‑friendly layout
* Fast loading

NO:

* Popups
* Fake buttons
* Forced redirects

## 18) Apply for Google AdSense

3. Verify ownership
4. Submit for review

⏱ Approval usually: **3–14 days**
